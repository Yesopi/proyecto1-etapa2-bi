¿Cómo funciona?

Este sistema utiliza un modelo de aprendizaje automático entrenado con miles de noticias para identificar patrones lingüísticos que pueden indicar si una noticia es potencialmente falsa.

El detector analiza el texto y proporciona una clasificación junto con la probabilidad de que la noticia sea verdadera o falsa.
